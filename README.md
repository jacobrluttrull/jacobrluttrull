## Selected Projects
 
### **Portfolio Website** — Full-Stack Personal Site
Full-stack portfolio site built with FastAPI and PostgreSQL, deployed on Railway. Features a JWT-protected admin panel for managing projects, a contact form with XSS sanitization, CSRF protection, Cloudflare Turnstile CAPTCHA, rate limiting, and Gmail notifications. Includes security middleware (CSP, HSTS, X-Frame-Options), Alembic migrations, rotating file logging, and a pytest test suite.
 
- Live: [jacobrluttrull.com](https://jacobrluttrull.com)
- GitHub: [jacobrluttrull/portfolio](https://github.com/jacobrluttrull/portfolio)
 
### **Static Site Generator** — Custom Markdown-to-HTML Generator
Python-based static site generator built from scratch that converts Markdown content into a fully static HTML/CSS website. Features recursive directory traversal, a custom Markdown parser (no external libraries), template-based HTML generation, and configurable base paths for deployment. Deployed live on GitHub Pages.
 
- Live Demo: [jacobrluttrull.github.io/static_site_generator](https://jacobrluttrull.github.io/static_site_generator/)
- GitHub: [jacobrluttrull/static_site_generator](https://github.com/jacobrluttrull/static_site_generator)
 
### **Oklahoma Sooners Dashboard** — College Football Statistics Dashboard
Full-stack Django web app integrating the CollegeFootballData (CFBD) API to surface season records, rankings context, player leaders, a ranked schedule, and live game box scores. Normalized relational schema with idempotent schedule sync and freshness-based caching.
 
- GitHub: [jacobrluttrull/oklahoma_sooners_dashboard](https://github.com/jacobrluttrull/oklahoma_sooners_dashboard)
 
### **AI_Bot** — Tool-Using AI Coding Agent (REPL + Function Calling)
Python-based CLI AI assistant using the OpenAI API with function calling and a multi-step agent loop. Supports one-shot prompts and an interactive REPL mode, with tools for listing files, reading/writing files, running Python scripts, and executing pytest test suites.
 
- GitHub: [jacobrluttrull/AI_Bot](https://github.com/jacobrluttrull/AI_Bot)
 
### **Diabetic Prediction Website** — Diabetes Risk Prediction App
Machine learning application using XGBoost and scikit-learn to predict Type 2 diabetes risk with ~80% accuracy. Includes an interactive Streamlit web app with SHAP visualizations for explainable predictions.
 
- GitHub: [jacobrluttrull/prod-diabetic-capstone](https://github.com/jacobrluttrull/prod-diabetic-capstone)
 
### **python_git** — Build Your Own Git (Python)
Hands-on implementation of core Git internals in Python, exploring how version control works under the hood.
 
- GitHub: [jacobrluttrull/python_git](https://github.com/jacobrluttrull/python_git)
